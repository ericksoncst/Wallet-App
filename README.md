# Wallet App

How to run that project:

```bash
git checkout master
```

Package instalation:

```bash
yarn
```
or 

```bash
npm i
```

## Starting Json Server

My commits already have a db.json. Follow that config:

```bash
json-server --watch db.json -p 3000 -d 2000 --host [yourIp]
```

## Runing Project

```bash
yarn start
```


```bash
yarn android
```
